Мой первый консольный калькулятор на C++.


 ## Возможности:
 -сложение;
 -вычитание;
 -деление;
 -умножение;
 -поддержка дробных чисел.

 ##Сборка и установка
 -для сорки выполните команду make
 -после сборки sudo make install
 -для удаления sudo make uninstall

 -после установки просто наберите в терминале calc.

 ##Зависимости

 -Компилятор C++ (g++ || clang+)
 -Make
